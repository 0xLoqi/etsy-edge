How does Tag Spy find related searches?

Etsy surfaces related search terms for every listing through publicly available data on the page. Tag Spy reads this data and displays it in a convenient side panel while you browse ‚Äî no scraping or private data involved.

Does it work on all Etsy listings?

Yes. Etsy Edge works on every public Etsy listing page. Just navigate to any listing and click the extension icon ‚Äî the side panel opens right next to the page.
