Privacy Policy

Last updated: February 2026

What Etsy Edge Does

Etsy Edge is a browser extension that helps Etsy sellers optimize their listings for search. It reads publicly available data from Etsy listing pages and the Etsy Open API to display tags, optimization scores, and smart suggestions.

Data We Collect

We do not collect, store, or log any personal data.

Specifically:

  • No API keys required. All API keys are held securely on our backend server (a Cloudflare Worker). You never need to obtain or enter any API keys.
  • Listing data (tags, titles, optimization scores) is fetched via our backend proxy which forwards requests to the Etsy API. Listing data passes through our server but is never stored or logged.
  • Smart suggestions are generated by sending listing titles and descriptions through our backend proxy to a third-party language model API. This data passes through our server but is never stored or logged.
  • Payment data is handled entirely by ExtensionPay/Stripe. We never see or store your payment information.

Third-Party Services

  • Cloudflare Workers — Our backend proxy runs on Cloudflare Workers. Requests are forwarded to Etsy and Anthropic APIs. No request data is stored or logged. Subject to Cloudflare's Privacy Policy.
  • Etsy Open API — Used to fetch listing data (tags, titles, search results). Subject to Etsy's API Terms of Use.
  • Anthropic API — Used for smart tag suggestions (Pro feature only). Subject to Anthropic's Usage Policy.
  • ExtensionPay / Stripe — Used for subscription payments. Subject to Stripe's Privacy Policy.

Data Storage

Locally in your browser (Chrome Storage API):

  • Cached tag results (auto-expires after 1 hour)
  • User preferences (toggle settings)

No data is stored on our servers. You can clear all local data by removing the extension from Chrome.

Permissions

  • storage — To save your settings and cache API results locally.
  • activeTab — To read the current Etsy listing page and display the Tag Spy overlay.
  • Host permission (etsy.com) — To inject the content script on Etsy listing pages.

Children's Privacy

Etsy Edge is not directed at children under 13. We do not knowingly collect data from children.

Changes to This Policy

We may update this policy from time to time. Changes will be reflected in the "Last updated" date above.
