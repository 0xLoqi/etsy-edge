Privacy Policy

Last updated: February 2026

What Etsy Edge Does

Etsy Edge is a browser extension that helps Etsy sellers optimize their listings for search. It reads publicly available data from Etsy listing pages and the Etsy Open API to display tags, SEO scores, and optimization suggestions.

Data We Collect

We do not collect, store, or log any personal data.

Specifically:

  • No API keys required. All API keys are held securely on our backend server (a Cloudflare Worker). You never need to obtain or enter any API keys.
  • Listing data (tags, titles, SEO scores) is fetched via our backend proxy which forwards requests to the Etsy API. Listing data passes through our server but is never stored or logged.
  • AI suggestions are generated by sending listing titles and descriptions through our backend proxy to OpenAI's API. This data passes through our server but is never stored or logged.
  • Payment data is handled entirely by ExtensionPay/Stripe. We never see or store your payment information.

Third-Party Services

  • Cloudflare Workers ‚Äî Our backend proxy runs on Cloudflare Workers. Requests are forwarded to Etsy and OpenAI APIs. No request data is stored or logged. Subject to Cloudflare's Privacy Policy.
  • Etsy Open API ‚Äî Used to fetch listing data (tags, titles, search results). Subject to Etsy's API Terms of Use.
  • OpenAI API ‚Äî Used for AI-powered tag suggestions (Pro feature only). Subject to OpenAI's Terms of Use.
  • ExtensionPay / Stripe ‚Äî Used for subscription payments. Subject to Stripe's Privacy Policy.

Data Storage

Locally in your browser (Chrome Storage API):

  • Cached tag results (auto-expires after 1 hour)
  • User preferences (toggle settings)

No data is stored on our servers. You can clear all local data by removing the extension from Chrome.

Permissions

  • storage ‚Äî To save your settings and cache API results locally.
  • activeTab ‚Äî To read the current Etsy listing page and display the Tag Spy overlay.
  • Host permission (etsy.com) ‚Äî To inject the content script on Etsy listing pages.

Children's Privacy

Etsy Edge is not directed at children under 13. We do not knowingly collect data from children.
