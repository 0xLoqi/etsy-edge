Etsy Edge — Privacy Policy

Last updated: February 2026

What Etsy Edge Does

Etsy Edge is a browser extension that helps Etsy sellers optimize their listings for search. It reads publicly available data from Etsy listing pages and the Etsy Open API to display tags, SEO scores, and optimization suggestions.

Data We Collect

We do not collect, store, or transmit any personal data.

Specifically:

  • API keys (Etsy, OpenAI) are stored locally in your browser using Chrome's built-in storage API. They are never sent to our servers — they are only sent directly to the respective API services (Etsy, OpenAI) from your browser.
  • Listing data (tags, titles, SEO scores) is fetched from the Etsy API and processed entirely within your browser. No listing data is sent to or stored on our servers.
  • AI suggestions are generated by sending listing titles and descriptions directly from your browser to OpenAI's API. We do not proxy or store these requests.
  • Payment data is handled entirely by ExtensionPay/Stripe. We never see or store your payment information.

Third-Party Services

  • Etsy Open API — Used to fetch listing data (tags, titles, search results). Subject to Etsy's API Terms of Use.
  • OpenAI API — Used for AI-powered tag suggestions (Pro feature only). Subject to OpenAI's Terms of Use.
  • ExtensionPay / Stripe — Used for subscription payments. Subject to Stripe's Privacy Policy.

Data Storage

All data is stored locally in your browser using the Chrome Storage API:

  • API keys (encrypted by Chrome's storage)
  • Cached tag results (auto-expires after 1 hour)
  • User preferences (toggle settings)

You can clear all stored data by removing the extension from Chrome.

Permissions

  • storage — To save your settings and cache API results locally.
  • activeTab — To read the current Etsy listing page and display the Tag Spy overlay.
  • Host permission (etsy.com) — To inject the content script on Etsy listing pages.

Children's Privacy

Etsy Edge is not directed at children under 13. We do not knowingly collect data from children.
