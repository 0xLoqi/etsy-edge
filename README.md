## How It Works

1. Install the extension from the Chrome Web Store
2. Browse any Etsy listing page
3. The Etsy Edge panel appears in the bottom-right corner with tags, SEO score, and more

The extension reads publicly available data from Etsy listing pages and the Etsy Open API. All processing happens locally in your browser.

## Tech Stack

- **Framework**: [WXT](https://wxt.dev) (Vite-based Chrome extension framework)
- **UI**: React 19 + TypeScript
- **Styling**: Tailwind CSS v4
- **AI**: OpenAI GPT-4o-mini
- **Payments**: ExtensionPay (Stripe)
